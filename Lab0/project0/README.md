# Project 0 - Lab 0

## Project Overview
This project is a personal portfolio website created as part of **ENGO 651 – Lab 0**.  
The goal of this assignment was to become comfortable with **HTML, CSS, SCSS, and Bootstrap 4** by designing a responsive multi-page website.

The website presents information about my academic background, skills, experience, and contact details.

---

## Pages Included
- **index.html** – Home page introducing myself and the purpose of the website  
- **education.html** – Academic background including Master of Engineering (University of Calgary) and Bachelor of Engineering (Gujarat Technological University)  
- **experience.html** – Professional experience  
- **about.html** – Personal summary, interests, and skills table  
- **contact.html** – Contact form and social/contact links  

All pages are connected through a common navigation bar.

---

## Technologies Used
- **HTML5**
- **CSS3**
- **SCSS** (compiled to CSS)
- **Bootstrap 4**
- **Git & GitHub**

---

## Styling and Design
- Custom styles are written in **style.scss** and automatically compiled to **style.css**
- SCSS features used:
  - Variables 
  - Nesting
  - Inheritance using `@extend`
- Responsive design implemented using **@media queries**
- Bootstrap grid system is used for layout and responsiveness

---

## Additional Notes
- The website is fully responsive and adapts to smaller screen sizes.
- Bootstrap components such as navbar, grid, forms, and tables are used.
